# ザ・ポモドーロタイマー
このアプリは、ポモドーロテクニックを利用する際に便利なタイマーアプリです。

## 特徴
- 集中しやすいデザイン
    - シンプルな機能・シンプルなデザインなので使いやすく集中できます。
- 細かいカスタマイズ
    - 作業時間だけでなく、セット数・休憩までの繰り返し回数など、細かいカスタマイズが可能です。
